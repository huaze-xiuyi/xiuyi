📖 使用指南

HTML 中使用文本色：

<h1 class="text-primary">标题</h1> <p class="text-secondary">副文本</p> <p class="text-tertiary">正文</p> <span class="text-muted">禁用</span>

CSS 中使用变量：

.my-component { background: var(--bg-secondary); color: var(--text-primary); border: 1px solid var(--border-color); }

  • 所有颜色在红绿色盲模式下清晰
  • 在移动设备上显示正常
