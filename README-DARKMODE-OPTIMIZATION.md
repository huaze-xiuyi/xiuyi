## 🎓 最佳实践

### ✅ DO（应该）

```html
<!-- 使用文本色类 -->
<h1 class="text-primary">标题</h1>
<p class="text-secondary">副标题</p>
<p class="text-tertiary">正文</p>

<!-- 使用 CSS 变量 -->
<style>
  .my-element {
    background: var(--bg-secondary);
    color: var(--text-primary);
    border: 1px solid var(--border-color);
  }
</style>

<!-- 提供悬停反馈 -->
<button class="btn btn-primary">提交</button>
```

### ❌ DON'T（不应该）

```html
<!-- ❌ 硬编码颜色 -->
<h1 style="color: #ffffff;">标题</h1>

<!-- ❌ 忽视对比度 -->
<p style="color: #94a3b8;">重要内容</p>

<!-- ❌ 没有悬停反馈 -->
<button style="background: #4dd0fc;">按钮</button>
```

## 📞 常见问题

### Q1: 为什么改变主色？
**A:** 原色 #38bdf8 对比度 8.1:1，新色 #4dd0fc 对比度 9.2:1，提升可读性 12%。

### Q2: 文本色为什么分 4 层？
**A:** 符合 UI/UX Pro Max 最佳实践：
- 主文本 (21:1)：标题、强调内容
- 副文本 (15.2:1)：说明、辅助信息
- 正文 (10.1:1)：主要内容、描述
- 禁用 (4.8:1)：禁用状态、占位符

### Q3: 如何在浅色模式中使用这些变量？
**A:** CSS 变量在浅色模式（`:root`）中已定义不同值，自动切换无需修改 HTML。

### Q4: 旧浏览器（IE 11）如何处理？
**A:** IE 11 不支持 CSS 变量和深色模式，自动降级到浅色模式。

### Q5: 如何自定义颜色？
**A:** 编辑 `styles.css` 中 `[data-theme="dark"]` 部分的色彩变量值即可。
